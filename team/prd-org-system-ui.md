## 一、设计原则

### 1.1 核心设计理念
- **极简主义**: 白色背景 + 极简图标，减少视觉干扰
- **卡片式布局**: 圆角 12px + 轻微阴影，层次分明
- **数据驱动**: 可视化优先，关键指标一目了然
- **响应式**: 适配桌面端主流分辨率 (1920x1080, 1366x768)

### 1.2 色彩规范
| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#F5F7FA` | 浅灰背景，降低视觉疲劳 |
| 卡片背景 | `#FFFFFF` | 纯白卡片 |
| 主色调 | `#409EFF` | PrimeVue 蓝，用于按钮/高亮 |
| 成功色 | `#67C23A` | 正常/在职状态 |
| 警告色 | `#E6A23C` | 待处理/试用期 |
| 危险色 | `#F56C6C` | 异常/离职状态 |
| 文字主色 | `#303133` | 主要文字 |
| 文字次要色 | `#909399` | 辅助文字 |
| 边框色 | `#EBEEF5` | 卡片/表格边框 |

### 1.3 字体规范
- **字体家族**: `Helvetica Neue, Helvetica, PingFang SC, Hiragino Sans GB, Microsoft YaHei, Arial, sans-serif`
- **标题**: 18px / 600 (页面标题), 16px / 600 (卡片标题)
- **正文**: 14px / 400
- **辅助文字**: 12px / 400

## 四、交互设计

### 4.1 导航交互
- Click 导航项: 切换主内容区，高亮当前项
- 支持键盘导航 (↑↓ 键切换，Enter 确认)

### 4.2 搜索交互
- 输入实时过滤 (防抖 300ms)
- 支持拼音首字母搜索
- 搜索结果高亮匹配关键词

### 4.3 表格交互
- 点击表头: 排序 (升序/降序/取消)
- 行 Hover: 显示操作按钮
- 行 Click: 弹出详情侧边栏
- 支持批量选择 (Checkbox)

### 4.4 图表交互
- 悬浮显示详细数据 (Tooltip)
- Click 图例: 显示/隐藏对应数据系列
- 支持缩放 (鼠标滚轮)

### 4.5 通知交互
- Click 通知图标: 弹出未读通知列表
- 通知分类: 系统通知、审批提醒、公告
- 标记已读: Click 单个通知或"全部已读"按钮

## 五、响应式设计

### 5.1 断点定义
| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| Desktop | ≥1440px | 3 列卡片，完整表格 |
| Laptop | 1024-1439px | 3 列卡片，表格隐藏 2 列 |
| Tablet | 768-1023px | 2 列卡片，表格转为卡片列表 |
| Mobile | <768px | 1 列卡片，隐藏左侧导航 (汉堡菜单) |

### 5.2 移动端适配
- 左侧导航: 折叠为汉堡菜单 (左上角)
- 顶部工具栏: 搜索框收缩为图标，Click 展开
- 仪表盘卡片: 单列垂直排列
- 数据表格: 转为卡片式列表 (每行一张卡片)

## 六、技术实现建议

### 6.1 技术栈
- **框架**: Vue 3.4+
- **UI 库**: PrimeVue 4.x
- **图表**: Chart.js / ECharts
- **状态管理**: Pinia
- **HTTP**: Axios

### 6.2 PrimeVue 组件映射
| 我们的组件 | PrimeVue 组件 | 自定义样式 |
|-----------|---------------|-----------|
| 左侧导航 | `Menu` / `TieredMenu` | 自定义激活态 |
| 顶部工具栏 | `Toolbar` | 自定义布局 |
| 卡片 | `Card` | 圆角 12px + 阴影 |
| 表格 | `DataTable` | 自定义表头/行样式 |
| 进度条 | `ProgressBar` | 自定义颜色 |
| 标签 | `Tag` / `Badge` | 自定义状态色 |
| 图表 | `Chart` (PrimeVue wrapper) | 自定义配色 |

### 6.3 性能优化
- 表格虚拟滚动 (超过 100 行时启用)
- 图表数据懒加载
- 图片/图标使用 SVG Sprite
- 路由懒加载
