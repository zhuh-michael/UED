# Input 输入框

基本的表单输入组件。

## 何时使用

- 需要用户输入文本信息
- 表单数据收集
- 搜索框

## 基础用法

```jsx
import { Input } from '@/components/ui'

<Input placeholder="请输入内容..." />
```

## 受控组件

```jsx
const [value, setValue] = useState('')

<Input 
  value={value} 
  onChange={(val) => setValue(val)} 
  placeholder="受控输入框"
/>
```

## 输入框尺寸

```jsx
<Input size="small" placeholder="小尺寸" />
<Input size="medium" placeholder="中尺寸（默认）" />
<Input size="large" placeholder="大尺寸" />
```

## 输入框状态

```jsx
<Input status="default" placeholder="默认状态" />
<Input status="success" placeholder="成功状态" />
<Input status="warning" placeholder="警告状态" />
<Input status="error" placeholder="错误状态" />
```

## 禁用状态

```jsx
<Input disabled placeholder="禁用状态" />
```

## 带前缀/后缀

```jsx
<Input prefix="¥" placeholder="金额" />
<Input suffix=".com" placeholder="域名" />
```

## API

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| value | 输入值 | `string` | - |
| placeholder | 占位符 | `string` | `'请输入内容...'` |
| size | 输入框尺寸 | `small` \| `medium` \| `large` | `medium` |
| disabled | 禁用状态 | `boolean` | `false` |
| status | 状态 | `default` \| `success` \| `warning` \| `error` | `default` |
| prefix | 前缀内容 | `ReactNode` | - |
| suffix | 后缀内容 | `ReactNode` | - |
| onChange | 变化事件 | `(value, e) => void` | - |
| onFocus | 聚焦事件 | `(e) => void` | - |
| onBlur | 失焦事件 | `(e) => void` | - |
| className | 自定义类名 | `string` | `''` |

## 设计说明

- 默认边框色：#E5E6EB
- 聚焦边框色：#3370FF
- 聚焦光晕：4px rgba(51, 112, 255, 0.2)
- 圆角：8px
- 过渡动画：0.3s ease
