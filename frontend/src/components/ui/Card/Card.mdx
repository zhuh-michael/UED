# Card 卡片

卡片用于分组和组织相关信息。

## 何时使用

- 展示一组相关的信息
- 作为列表项的容器
- 展示数据概览或统计信息

## 基础用法

```jsx
import { Card } from '@/components/ui'

<Card title="卡片标题">
  <p>这是卡片内容</p>
</Card>
```

## 卡片类型

### 玻璃态卡片

```jsx
<Card variant="glass" title="玻璃态卡片">
  半透明背景 + 背景模糊效果
</Card>
```

### 渐变卡片

```jsx
<Card variant="gradient" title="渐变卡片">
  蓝紫渐变背景，视觉冲击力强
</Card>
```

### 默认卡片

```jsx
<Card variant="default" title="默认卡片">
  白色背景，简洁样式
</Card>
```

## 可悬停交互

```jsx
<Card hoverable={true} title="可悬停卡片">
  悬停时上浮并增强阴影
</Card>
```

## 带额外内容

```jsx
<Card 
  title="卡片标题" 
  extra={<span>更多 →</span>}
>
  卡片内容
</Card>
```

## API

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| variant | 卡片类型 | `glass` \| `gradient` \| `default` | `glass` |
| title | 卡片标题 | `ReactNode` | - |
| extra | 右上角额外内容 | `ReactNode` | - |
| hoverable | 是否可悬停 | `boolean` | `true` |
| onClick | 点击事件 | `(e) => void` | - |
| className | 自定义类名 | `string` | `''` |

## 设计说明

- 圆角：12px
- 内边距：24px
- 悬停效果：上浮 4px + 阴影加深
- 玻璃态效果：backdrop-filter blur(20px)
