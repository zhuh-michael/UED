🔘 UED 基础组件库（4 个）

🔘 1. Button 按钮

主按钮
禁用状态

📖 使用说明

使用场景
操作触发、表单提交、功能入口
尺寸规范
  • 高度：36px（默认）/ 48px（大）/ 28px（小）
  • 内边距：0 24px
  • 圆角：8px
代码示例
<Button type="primary">提交</Button>
<Button disabled>禁用</Button>

📝 2. Input 输入框

默认状态
聚焦状态

📖 使用说明

使用场景
文本输入、表单填写、搜索框
尺寸规范
  • 高度：36px / 40px（大）
  • 内边距：0 12px
  • 圆角：8px
代码示例
<Input placeholder="请输入" />
<Input value={value} onChange={handleChange} />

📦 3. Card 卡片

📊 卡片标题
卡片内容区域，支持文字、图片、表格等。

📖 使用说明

使用场景
内容容器、数据展示、列表项
尺寸规范
  • Header 高度：48px（可选）
  • Body 内边距：24px
  • 圆角：12px
代码示例
<Card title="标题">
内容
</Card>

📄 4. Textarea 多行文本

📖 使用说明

使用场景
多行文本输入、评论、备注
尺寸规范
  • 高度：自适应（最小 80px）
  • 行数：4 行（默认）
  • 圆角：8px
代码示例
<Textarea rows={4} placeholder="备注" />
