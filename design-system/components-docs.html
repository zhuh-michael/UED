📖 使用说明

使用场景
操作触发、表单提交、功能入口
尺寸规范
  • 大按钮：48px × 32px，字体 16px（用于主要操作）
  • 中按钮：36px × 20px，字体 14px（默认尺寸）
  • 小按钮：28px × 12px，字体 12px（用于紧凑布局）
状态说明
状态 背景色 文字色 交互
默认 蓝紫渐变 white -
Hover 渐变 + 上浮 2px white 阴影增强
Active 加深 10% white 按下效果
Disabled #F5F5F5 rgba(0,0,0,0.25) 不可点击
Loading 渐变 + Spinner white 禁止交互

import { Button } from '@/components/ui' // 主按钮 <Button type="primary">提交</Button> // 次按钮 <Button type="secondary">取消</Button> // 加载状态 <Button loading>加载中...</Button> // 禁用状态 <Button disabled>不可用</Button>

📝 2. Input 输入框

默认状态
聚焦状态（Focus）
禁用状态
错误状态
⚠️ 此项为必填项

📖 使用说明

使用场景
文本输入、表单填写、搜索框
尺寸规范
  • 高度：36px（默认）/ 32px（小）/ 40px（大）
  • 内边距：0 12px
  • 字体：14px
  • 圆角：8px
交互说明
  • Focus：边框变主色 + 光晕 3px
  • Hover：边框主色 50% 透明度
  • Error：边框红色 + 错误提示文字
  • Disabled：灰色背景 + 不可编辑
代码示例
import { Input } from '@/components/ui' // 基础输入框 <Input placeholder="请输入" /> // 带验证 <Input value={value} onChange={handleChange} error={error} /> // 禁用状态 <Input disabled />
