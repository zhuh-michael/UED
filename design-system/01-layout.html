📐 1. Layout 布局容器

经典上下左右布局
Logo 导航菜单 | 用户头像
📊 仪表盘
📁 部门管理
👥 员工管理
📋 岗位管理

主内容区

页面主要内容区域，自适应宽度

© 2026 青鸾组织架构管理系统 | 技术支持

📖 使用说明

使用场景
后台管理系统主框架、企业应用整体布局
尺寸规范
  • Header 高度：60px
  • Sidebar 宽度：200px（支持折叠）
  • Footer 高度：40px
  • Content：自适应宽度，内边距 24px
结构说明
  • Layout：最外层容器，flex 纵向布局
  • Header：顶部导航，固定高度
  • Layout.Body：中间区域，flex 横向布局
  • Sidebar：左侧导航，固定宽度
  • Content：主内容区，flex: 1 自适应
  • Footer：底部信息，固定高度
代码示例
import { Layout } from '@/components/ui' <Layout> <Layout.Header> <Logo /> <NavMenu /> <UserAvatar /> </Layout.Header> <Layout.Body> <Layout.Sidebar> <Menu items={menuItems} /> </Layout.Sidebar> <Layout.Content> <RouterOutlet /> </Layout.Content> </Layout.Body> <Layout.Footer> © 2026 Company Name </Layout.Footer> </Layout>

📄 4. Content 内容区

📊 数据统计
内容区支持卡片式布局，可放置表格、表单、图表等内容组件。 背景色 #FAFAFA，与白色卡片形成对比。
📋 待办事项
支持多个内容卡片垂直排列，保持统一间距和视觉风格。 卡片间距 16px，圆角 12px，阴影轻微。

📖 使用说明

使用场景
页面主内容区域、数据展示、表单容器
尺寸规范
  • 宽度：自适应（flex: 1）
  • 内边距：24px
  • 背景色：#FAFAFA
  • 卡片间距：16px
布局建议
  • 单列布局：适合表单、详情
  • 双列布局：适合列表 + 详情
  • 网格布局：适合数据卡片
代码示例
import { Content, Card } from '@/components/ui' <Content padding={24}> <Card title="数据统计"> <StatisticsChart data={data} /> </Card> <Card title="待办事项" style={{ marginTop: 16 }}> <TodoList items={todos} /> </Card> </Content>

📦 8. Container 容器

固定宽度容器

max-width: 1200px，居中显示

📖 使用说明

使用场景
页面内容居中、限制最大宽度、统一内容区域
尺寸规范
  • 固定宽度：1200px（可配置）
  • 响应式：100%（<1200px）
  • 左右边距：auto（居中）
  • 内边距：24px
代码示例
import { Container } from '@/components/ui' // 固定宽度（居中） <Container width={1200}> <PageContent /> </Container> // 流体宽度（100%） <Container fluid> <FullWidthContent /> </Container>

📐 布局组件使用指南

1. 标准后台布局： Layout → Header + Sidebar + Content + Footer
2. 简单页面布局： Header + Content
3. 响应式布局： 使用 Grid 栅格系统 + Space 间距
4. 内容分隔： 使用 Divider 分割线
5. 内容居中： 使用 Container 容器
